# Quora Clone (REST) 📝

A **mini project** inspired by Quora, built with basic REST operations.  
Users can create, read, update, and delete posts.

## 🚀 Features
- Add new posts
- Edit existing posts
- Delete posts
- Simple REST API usage

## 🛠️ Tech Stack
- Node.js
- Express.js
- REST API concepts

## ▶️ Run Locally
```bash
npm install
node app.js
